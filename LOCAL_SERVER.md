## Start the Server

```bash
# 1. Navigate to the website-v1 folder
cd website-v1

# 2. Activate the virtual environment
source ../venv/Scripts/activate

# 3. Run the Flask app
python app.py
```

The server will start at: **http://127.0.0.1:5000**

---

## Stop the Server

Press **`Ctrl + C`** in the terminal where the server is running.

---

## Deactivate Virtual Environment (when done)

```bash
deactivate
```

---

## Quick One-Liner (from v1 folder)

```bash
cd website-v1 && source ../venv/Scripts/activate && python app.py
```
