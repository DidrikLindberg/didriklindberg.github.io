## üé® DESIGN SYSTEM

### Colors
- **Primary**: Blue/Purple gradient (#667eea to #764ba2)
- **Text**: Gray-900 (headings), Gray-700 (body)
- **Background**: White, Gray-50 for sections
- **Accents**: Green (success), Red (errors), Blue (links)

### Typography
- **Font**: Inter (Google Fonts)
- **Headings**: Bold, 2xl to 5xl sizes
- **Body**: Regular, lg to base sizes
- **Line height**: Relaxed (1.625)

### Components
- **Buttons**: Rounded-lg, shadow, hover scale
- **Cards**: Border, rounded-xl, hover shadow
- **Forms**: Border, rounded-lg, focus ring
- **Icons**: Heroicons (inline SVG)

### Spacing
- Sections: py-20 (80px vertical padding)
- Cards: p-8 (32px padding)
- Grids: gap-8 (32px gaps)

### Animations
- Hover scale on buttons
- Hover shadow on cards
- Smooth transitions (all 300ms)
- Arrow animations on links
